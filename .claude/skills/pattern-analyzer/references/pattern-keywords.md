# 정기 업무 패턴 키워드 사전

패턴 추론 시 아래 키워드가 태스크 제목/내용에 포함되면 정기 반복 가능성이 높다고 판단한다.

## 주기 키워드

| 카테고리 | 키워드 목록 |
|----------|------------|
| 일간 | 일일, 데일리, daily |
| 주간 | 주간, 주별, 위클리, weekly, 매주 |
| 격주 | 격주, 2주, biweekly |
| 월간 | 월간, 월별, 먼슬리, monthly, 매월, 매달 |
| 분기 | 분기, 분기별, quarterly |
| 연간 | 연간, 연별, 연도별, 매년, 연말, 연초 |

## 업무 유형 키워드

| 카테고리 | 키워드 목록 |
|----------|------------|
| 보고 | 보고, 리포트, report, 현황, 결과, 집계, 취합 |
| 회의/미팅 | 회의, 미팅, meeting, 스탠드업, standup, 싱크, sync |
| 리뷰 | 리뷰, review, 검토, 점검, 확인, 체크 |
| 정산/처리 | 정산, 처리, 마감, 마무리, 제출, 발행 |
| 계획 | 계획, 플랜, plan, 기획, 준비 |
| 모니터링 | 모니터링, 모니터, monitoring, 관제, 점검 |

## 판단 규칙

1. 위 키워드가 제목에 포함되어 있으면 **정기 업무 후보**로 분류한다.
2. 동일/유사 제목이 이력에 **2회 이상** 등장하면 반복 가능성이 높다.
3. 날짜 간격이 **7일, 14일, 30일, 90일** 근처(±2일 이내)이면 주기로 인정한다.
4. 키워드만 있고 날짜 데이터가 없으면 신뢰도를 **40–60%**로 제한한다.
5. 신뢰도 **50% 미만** 항목은 브리핑에 "(추정, 확인 필요)" 태그를 붙인다.
